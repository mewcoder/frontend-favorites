# 前端摸鱼周刊

- 本周刊主要用于收集摸鱼时看到的优质的内容，比如优秀的文章或专栏，有用的工具或者网站，等等
- 通过日积月累的收集和不同的标签进行知识管理，将碎片化的信息积累成一个前端知识/工具库。
- 本周刊不强调时效性，但是内容必须要优质、有用。


日拱一卒，欢迎投稿[提交 issue](https://github.com/fe-focus/moyu-weekly/issues/new/choose) 。
